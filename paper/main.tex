\begin{abstract}
Function vectors (FVs; Todd et al., 2024) are compact task representations
extracted from a small set of attention heads that, when added to the residual
stream, induce zero-shot task execution. We reproduce the core FV results on
GPT-J-6B and test their generalizability to three modern grouped-query-attention
models: Llama-3.1-8B, Qwen2.5-7B, and Mistral-7B-v0.3. % TODO: results sentence
\end{abstract}
